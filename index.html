Project Overview

A mobile app prototype designed to help users plan meals, track nutrition, and manage grocery lists based on dietary goals. Designed entirely in Figma using Material Design 3 components.

Design Process

  • Research: Surveyed users to understand habits around meal planning and nutrition tracking.
  • User Flows: Built 9 decision paths using Figma logic to simulate checkbox-based onboarding personalization.
  • Wireframes & Prototyping: Created high-fidelity screens and interactive flows for onboarding, meal selection, and grocery lists.
  • UI Design: Applied Material Design 3 components to ensure consistency, accessibility, and a clean aesthetic.
  • Presentation: Delivered a clickable prototype with scenario-based storytelling and annotated decision points.
